Browserbasiertes Tool zum Auswerten von Wettkämpfen

Speicherung lokaler Einstellungen:

Kopieren der Datei /mysite/settings.py und speichern als ../settings_local.py
Kopieren der Datei /grading/settings.xml und speichern als ../settings_local.xml
Solange diese Dateien vorhanden sind, greift das Tool auch auf diese zu und ignoriert die ursprünglichen Einstellungsdateien
